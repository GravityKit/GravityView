gravityview_get_entry()

gravityview_get_entry(integer|string  $entry_slug, boolean  $force_allow_ids = false, boolean  $check_entry_display = true, \GV\View  $view = null) : array|boolean

Return a single entry object

Since 1.4, supports custom entry slugs. The way that GravityView fetches an entry based on the custom slug is by searching gravityview_unique_id meta. The $entry_slug is fetched by getting the current query var set by is_single_entry()

Parameters

integer|string $entry_slug

Entry ID or slug
boolean $force_allow_ids

Force the get_entry() method to allow passed entry IDs, even if the gravityview_custom_entry_slug_allow_id filter returns false.
boolean $check_entry_display

Check whether the entry is visible for the current View configuration. Default: true {@since 1.14}
\GV\View $view

The View if $check_entry_display is set to true. {@since develop}

Returns

array|boolean

File
