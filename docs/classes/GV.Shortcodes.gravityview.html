parse_and_sanitize_atts()

parse_and_sanitize_atts(array  $passed_atts) : array

Validate attributes passed to the [gravityview] shortcode. Supports {get} Merge Tags values.

Attributes passed to the shortcode are compared to registered attributes \GV\View_Settings::defaults Only attributes that are defined will be allowed through.

Then, {get} merge tags are replaced with their $_GET values, if passed

Then, attributes are sanitized based on the type of setting (number, checkbox, select, radio, text)

Parameters

array $passed_atts

Attribute pairs defined to render the View

Returns

array —

Valid and sanitized attribute pairs
